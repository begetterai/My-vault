# Бот-агент Ромашки — архитектура

Зафиксировано 14.07.2026. Цель: Азиз кидает голосовое/текст в Telegram — агент сам всё делает.

## Поток

```
Азиз (голос/текст) → Telegram-бот @Rabotyaga7_bot
   → [голос] скачивание файла → расшифровка (Groq Whisper)
   → Мозг (LLM с инструментами) понимает намерение
   → Выполняет нужный инструмент
   → Отвечает Азизу в Telegram + пишет в аудит-лог
```

## Компоненты

| Слой | Чем | Статус |
|---|---|---|
| Вход/выход | Telegram Bot API (long-polling) | ✅ токен есть |
| Расшифровка голоса | Groq Whisper (whisper-large-v3), рус. | нужен Groq-ключ |
| Мозг | LLM с tool-calling: Groq (free) или Anthropic (лучше) | нужен ключ |
| Действия | Notion API, Google Sheets/Drive, Poster API | ✅ подключены |
| Хостинг 24/7 | Railway | конфиг готов |
| Аудит | лог всех сообщений и действий в Google Sheet | строим |

## Инструменты агента (что он умеет делать сам)

1. **add_task** — добавить задачу. Поля: текст, тип (тактическая/стратегическая), кто, срок. → база Задачи в Notion.
2. **add_violation** — записать нарушение. Поля: точка, сотрудник, описание, категория. → база Нарушения в Notion.
3. **get_revenue** — выручка за день/неделю/месяц/год. Читает Google Sheet. → ответ текстом.
4. **poster_query** — вопрос по Poster (расходы/выручка по категории за период). → считает, отвечает.
5. **capture_note** — просто сохранить мысль/идею. → 0-Входящие в Vault (разберём в сессии).

Мозг сам выбирает инструмент по смыслу сообщения. Если непонятно — переспрашивает, не гадает.

## Правила надёжности (без дыр)

- **Безопасность:** агент отвечает ТОЛЬКО на chat_id Азиза (854893099). Чужой — «нет доступа».
- **Не теряем:** каждое сообщение и каждое действие пишется в аудит-лог (Google Sheet). Ничего не пропадает молча.
- **Честность действий:** если действие не удалось — агент говорит прямо, не делает вид, что сделал.
- **Подтверждение неоднозначного:** «записать нарушение на кого?» — если не указано. Явное — делает сразу и подтверждает.
- **Идемпотентность:** offset Telegram сохраняется, одно сообщение не обрабатывается дважды.
- **Отказоустойчивость:** ошибка сети/API → повтор с паузой; если совсем упало — пишет Азизу, не молчит.
- **Ограничение доступа Poster:** только чтение (аналитика/финансы), никаких изменений в кассе.

## Что НЕ входит (честно)

- **Instagram** — официального доступа к личным сообщениям нет. Вход только Telegram.
- **Автосписания/платежи** — агент не трогает деньги, только читает и фиксирует.

## Что нужно от Азиза

1. **Groq API-ключ** (бесплатно, groq.com) — покрывает и расшифровку голоса, и мозг.
2. (Опционально) **Anthropic API-ключ** — если хотим мозг умнее (платно, центы за сообщение).
3. Деплой на Railway (по инструкции).

## Этапы

1. **MVP:** голос/текст → задачи + нарушения + выручка. (строим первым)
2. **+ poster_query:** «сколько потратили на аренду в июне».
3. **+ capture_note** и алерты (выручка упала, поставка не закрыта).
