# Стандарты оформления документов — Ромашка

#project/romashka #area/career

---

## Google Docs

| Параметр | Значение |
|----------|----------|
| Шрифт | Times New Roman |
| Размер | 14 пт |
| Межстрочный интервал | Полуторный (1.5) |
| Выравнивание | По ширине |
| Абзацный отступ | 1,25 см (не пробелами) |
| Поле левое | 30 мм (3 см) |
| Поле правое | 10–15 мм (1–1.5 см) |
| Поле верхнее | 20 мм (2 см) |
| Поле нижнее | 20 мм (2 см) |
| Ориентация | Книжная, A4 |
| Заголовки | Жирный, без точки, 18 пт |

---

## Google Sheets

### Структура данных
- Заголовки столбцов обязательны
- Нет пустых строк и столбцов внутри таблицы
- Один тип данных в одном столбце
- Заголовки уникальны

### Внешний вид
| Параметр | Значение |
|----------|----------|
| Шрифт | Times New Roman |
| Размер | 14 пт |
| Текстовые данные | Выравнивание по левому краю |
| Числовые данные | Выравнивание по правому краю |
| Заголовки | По центру, жирный, 18 пт |
| Закрепление строк | Верхняя строка (заголовки) |
| Ширина столбцов | По содержимому |

---

## Применение

Все новые документы создаются с этими стандартами автоматически через скрипты.
Существующие документы — обновляются по мере работы.
