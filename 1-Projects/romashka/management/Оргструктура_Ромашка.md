# Ромашка — Организационная структура

**Обновлено:** Май 2026  
**Формат:** 2 точки, линейно-функциональная структура

---

## Схема

```
┌─────────────────────────────────┐
│         УЧРЕДИТЕЛИ              │
│    (стратегия, дивиденды)       │
└────────────┬────────────────────┘
             │
┌────────────▼────────────────────┐
│    ОПЕРАЦИОННЫЙ ДИРЕКТОР        │
│    Азиз                         │
│    (операции, финансы, контроль)│
└──────┬──────────────────┬───────┘
       │                  │
┌──────▼──────┐    ┌──────▼──────┐
│  УПРАВЛЯЮЩИЙ│    │  УПРАВЛЯЮЩИЙ│
│  ЗБ         │    │  ОВИР       │

       │                  │
  ┌────┴────┐        ┌────┴────┐
  │  КУХНЯ  │        │  КУХНЯ  │

  │  КАССА  │        │  КАССА  │
  └─────────┘        └─────────┘
```

## Роли и зоны ответственности

### Учредители
- Стратегические решения (открытие/закрытие точек, крупные инвестиции)
- Утверждение бюджета и дивидендов
- Не участвуют в операционной деятельности

---

### Операционный директор — Азиз
**Зона:** вся сеть

| Функция | Частота |
|--------|---------|
| Контроль финансовых показателей (P&L, Cash Flow) | Ежемесячно |
| Контроль дневного трекера | Ежедневно (5 мин) |
| Оценка работы управляющих по KPI | Еженедельно |
| Утверждение крупных закупок (> лимита) | По запросу |
| Переговоры с поставщиками и арендодателями | По необходимости |
| Налоги, юридические вопросы, банки | По необходимости |
| Развитие: новые точки, маркетинг, системы | Стратегически |

---

### Управляющий точки
**Зона:** одна точка, полная операционная ответственность

#### Ежедневные обязанности
| Время | Задача |
|-------|--------|
| Открытие | Чеклист открытия (касса, оборудование, запасы) |
| Смена | Контроль кухни и зала, решение вопросов гостей |
| Закупки | Оприходование поставок в Poster, проверка накладных |
| Закрытие | Закрытие смены в Poster, подсчёт кассы, инкассация |
| 23:00 | Отчёт директору (см. шаблон) |

#### Еженедельные обязанности
- Инвентаризация остатков (кухня + бар)
- Планирование закупок на следующую неделю
- Оценка работы персонала

#### Ежемесячные обязанности
- Сверка P&L с директором
- Расчёт и подача ведомости по ФОТ
- Отчёт по расходам (OpEx) с документами

#### Полномочия и лимиты
| Действие | Лимит без согласования |
|---------|----------------------|
| Разовая закупка | до 2 000 с |
| Хозтовары, расходники | до 500 с |
| Скидка гостю | до 10% |
| Выход за лимит | → согласовать с директором |

---

### Производственный персонал (повара, кассиры, официанты)
- Выполнение стандартов приготовления и обслуживания
- Соблюдение санитарных норм
- Подчиняются управляющему точки
- Фиксация нарушений → управляющий

---

## KPI управляющего

| Показатель | Цель | Источник |
|-----------|------|---------|
| Выручка месяц | По плану точки | Дневной трекер |
| Food Cost | < 30% | Super P&L |
| Labor Cost | < 25% | Super P&L |
| EBITDA | > 15% | Super P&L |
| Расхождение кассы (J) | = 0 | Дневной трекер |
| Своевременность отчёта | Ежедневно до 23:00 | WhatsApp |
| Инвентаризация | Еженедельно | Акт |

---

## Коммуникация

| Канал | Назначение |
|-------|-----------|
| WhatsApp | Ежедневный отчёт, срочные вопросы |
| Встреча 1:1 | Еженедельно, 30 мин (онлайн или очно) |
| Сверка P&L | Раз в месяц, очно |
| Общий чат | Объявления, общие вопросы |

---

## Инструменты

| Инструмент | Кто использует |
|-----------|---------------|
| Poster POS | Управляющий + персонал |
| Дневной трекер (Google Sheets) | Управляющий вводит, директор смотрит |
| Super P&L (Google Sheets) | Директор, управляющий — сверка |
| WhatsApp | Вся команда |

---

## Найм и онбординг

- Управляющего нанимает директор
- Персонал нанимает управляющий (согласует с директором)
- Испытательный срок: 1 месяц
- Оценка после испытательного срока по KPI
