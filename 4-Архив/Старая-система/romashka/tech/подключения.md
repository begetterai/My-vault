## Скрипты

| Файл | Что делает | Запуск |
|------|-----------|--------|
| `scripts/poster_report.py` | Ежедневный / недельный / месячный отчёт (markdown + Drive) | `python3 scripts/poster_report.py day\|week\|month` |
| `scripts/sheets_report.py` | P&L + по дням + по неделям в Google Sheets | `python3 scripts/sheets_report.py 2026-03` |
| `scripts/poster_analytics.py` | Аналитика: KPI, маржинальность, продажи, поставки | `python3 scripts/poster_analytics.py 2026-03` |
| `scripts/drive_upload.py` | Загрузка любого файла в Drive (используется внутри) | `import drive_upload; drive_upload.upload(path)` |

## Данные из Poster (доступно сейчас)

### ✅ Финансы
- Все транзакции (доходы/расходы, категории, счета, даты)
- Категории финансовых операций (35 шт.)
- Счета и остатки: Касса, Сейф, Расчётный счёт, Инкассатор, Монеты
- Кассовые смены: открытие/закрытие, наличка, безнал, инкассация

### ✅ Меню
- 143 блюда с ценой, себестоимостью, прибылью и маржой
- 22 категории меню
- 148 ингредиентов

### ✅ Аналитика
- Выручка по дням, по часам (0–23), по дням недели
- Средний чек, количество чеков, посетители, среднее время
- Сравнение с предыдущим периодом

### ✅ Склад
- Поставки: поставщик, сумма, дата, склад, комментарий
- 4 склада: Кухня, Бар, Персоналка, Расходные материалы
- Списания: дата, сумма, причина (Порча и др.)

### ✅ Клиенты и персонал
- База клиентов: имя, телефон, бонусы, скидка, сумма покупок
- 21 сотрудник: имя, роль, последний вход

### ❌ Недоступно (нужен токен приложения)
- Продажи по блюдам (топ блюд по штукам)
- Остатки ингредиентов на складе
- Перемещения между складами
- Детали кухни (время приготовления)

## Возможности Drive/Sheets

- Автоматически создавать таблицы в папке **Ромашка**
- Обновлять существующие файлы (не дублировать)
- Форматирование: цвета, заморозка строк, ширина колонок
- Несколько листов в одном файле
